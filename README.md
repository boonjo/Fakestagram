<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/boonjo/Fakestagram.git
   ```
2. Run with Python
   ```bash
    python fake_insta.py
    ```
    or
    ```bash
    python3 fake_insta.py
    ```
